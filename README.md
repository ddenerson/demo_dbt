# Projeto exemplo dbt

Este repositório contém um projeto de demonstração do dbt com modelos, macros e configurações de seeds. O diretório `exemplo_dbt/` reúne todos os artefatos do projeto e pode ser usado como ponto de partida para explorar a ferramenta.

Consulte o README dentro de `exemplo_dbt/` para obter instruções detalhadas de configuração e execução.
